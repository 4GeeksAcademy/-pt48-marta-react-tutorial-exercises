# `01.6` Mapping array to `<li>`

Now let's build an array dynamically. Let's say you have an array of animals:

```js
const animals = [ 'Horse', 'Turtle', 'Elephan', 'Monkey' ];
```

# :speech_balloon: Instructions

Use the [.map function](https://medium.com/poka-techblog/simplify-your-javascript-use-map-reduce-and-filter-bd02c593cc2d) to create a new array of `<li>`'s that each of them corresponds one animal form the original array, the resulting array should be something like:

```jsx
const animalsInHTML = [
  <li>Horse</li>,
  <li>Turtle</li>,
  <li>Elephan</li>,
  <li>Monkey</li>
];
```
